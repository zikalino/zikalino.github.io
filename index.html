Module Azure_rm_virtualmachine - Delta

I have done some research on azure_rm_virtualmachine module. It’s one of the oldest modules. I went through all the properties supported by Azure REST API and compared with support available via module. I paid attention to: - what is currently supported - tested all the properties, whether they are updatable - checked idempotency supprt in the module
